Foodquest Database

Foodquest is a database created for SI 206. I used the RecipePuppy API to get a list of ingredients from a variety of recipes. We used the Open Food Facts API to determine how healthy each ingredient was and gave it a score from 1-5 and the Mapquest API to plot the countries that the healthiest ingredients were sold at.
